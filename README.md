# Hytale Mod: Feran Longtooth Merchant (NPC Shops And Trading)

Companion mod for the [Hytale Modding Manual](https://hytale-moddings.github.io/hytale-modding-docs/) tutorial on **NPC Shops and Trading**.

**Repository:** [hytale-moddings/hytale-mods-custom-shop](https://github.com/hytale-moddings/hytale-mods-custom-shop)

## What This Mod Does

Adds a **Feran Longtooth Enchanted Merchant** — an NPC that trades **Enchanted Fruit** for **Enchanted Saplings**. Demonstrates the shop/trading interaction schema.

## Installation

1. Copy the mod folder to `%APPDATA%/Hytale/UserData/Mods/NPCShopsAndTrading/`
2. The folder name **must** be `NPCShopsAndTrading`
3. Start a Hytale server with the mod enabled

## Testing In-Game

```
/spawnnpc HytaleModdingManual:FeranLongtoothMerchant
```

Right-click the merchant to open the trade UI.
